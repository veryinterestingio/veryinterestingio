MyNameIsNotPat A:

You are asking two possible questions.

How is a barcode made (a string of graphics which translates to a number or a number and text).

How is a product identification code (upc in the US & ean everywhere else issued)

A barcode is just an algorithm to show text in a different way, there are several algorithms. Plenty of free online barcode creators if you want to play around.

A product identification code has 4 components, country, company, product code & a checksum. If I want to put a ean on my product, I apply to my the issuer in my country and am given a block of numbers 941(NZ), 6932 (my company code, I made this up) and the rest of the digits are for me to use as I wish. No-one else can use ean codes starting with 9416932, so as long as I have my act together, there will be no clashing.

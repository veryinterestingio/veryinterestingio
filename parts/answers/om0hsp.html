konwiddak A:

So it depends.

You are always using energy at rest, simply to keep your body functioning and warm. Standing uses 100-200 calories per hour.

Walking somewhere at a comfortable pace uses about 200 calories per hour.

Jogging uses 400-700 calories per hour.

So you need to sum up the baseline calories that you always burn over time + the exertion calories and depending on the relative speed it could be:

  1. About the same if you're not working too hard running. You'll burn a bit more running, but not loads more.

  2. More from running if you're running fast.

  3. More from walking if you walk so slowly that the baseline calories from the time are significant. (E.g you walk for 10 hours something you can run in 30 minutes).

Running of course has other benefits:

If you only have an hour to exercise - you'll burn a lot more in the hour running than walking.

Running is good from a cardiovascular health point of view.

OP, when talking energy balances, you must look at DIFFERENCES, and remove the baseline.

If you spend 160W normally and 230W on an exercise and 300W on a heavier exercise, the difference is 70W for the 1st, 140W for the 2nd. So the real calorie difference from one to the other is double.

Ignore the baseline as that's fixed costs, sort of.

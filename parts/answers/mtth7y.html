It is likely these are the testing conditions it has been approved under, and so that is what they must say on the box, despite it being potentially safe to remove for longer. Safety and effecacy testing is expensive, so if it doesn't add value then they will go with the minimum test conditions for approval. Using it outside of what is stated would be using it 'off label', which is common for many pharmaceuticals, but means that the company is not liable if it fails when used inappropriately.

For example, the Mirena hormone coil has been shown in research studies to be effective for up to 7yrs, where as it is only licenced for up to 5yrs, as these are the testing conditions it was approved under. The company wouldn't make more sales by paying for expensive testing to extend use for another 2yrs, so using longer is off-label but likely safe and still effective.

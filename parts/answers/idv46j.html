Radiatin A:

What you're describing is accomplished using the magic technique of educated guessing. It's not physically possible to know what color something is certainly supposed to be simply from the data in a black and white image or video, but you can use clues related to context of each object to determine what color it should be such as uniforms from a specific period typically being one color. This guessing process can be done by humans or machines, it's ultimately the same fundamental concept of using context.

Our brains are fairly lazy so even if there are numerous errors in this transformation we aren't bothered by them.

If you're asking how technically it's possible to change a gray pixel to a colorized one, the gray tones are just mapped onto some selected shade of a color. So dark gray becomes dark red, light gray becomes light red etc within some boundary. The objects and their components are just separated by a set of boundaries which determine what color they should be.

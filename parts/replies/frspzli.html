    • MedusasSexyLegHair

      Yeah, that was a problem in the tabletop games too. If the winner ends up in a better position, so that it's easier for them to win the next match and so on then it snowballs and isn't fun.

      Imperium had a neat mechanic for that. During the peacetime between engagements, the winner's government would be more confident and lax, so they'd stand down more of the military forces, while the loser's government would be determined not to let that happen again, so they could preserve more force and end up more evenly-balanced for the next game. So winning a game put you in a better position strategically (more colonies/worlds conquered) but tactically, you'd start the next game with a disadvantage. You had to balance that over the campaign. It's rare where that sort of thing would fit into a game's design though.

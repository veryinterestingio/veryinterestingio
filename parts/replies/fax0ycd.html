    On PC (and on console, if you’re downloading a digital game), the initial download can bundle all the updates together with the base game and install them at once. On a console, when installing from a disc, the disc is only the base game (it’s physically printed and distributed that way), so only the base game installs. Then it has to go online to download and install updates.

      That's not what the question is asking though? If heat were to move from the colder source to the hotter source then the heating source would get hotter (Object A), not the colder object (Object B)? The reason this doesn't work is because of the first law of thermodynamics which says, "the internal energy of a thermodynamic system must be equal to the heat added to it minus the work being done by the system" so if Object B were to get hotter that would suggest thermal energy was being created from nothing, which as we know is impossible, because energy can not be created nor destroyed.

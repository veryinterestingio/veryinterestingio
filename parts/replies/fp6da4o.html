    This is the answer. In short, newspaper comics began to use Z's as visual shorthand for sleeping. Comics did the same thing with a lot of other symbols, creating a visual shorthand that we all know now. For example, thought balloons, a light bulb above a person's head, characters like #$*! to symbolize curse words, and so on.

    • vilhelm_s

      Current VR headsets do not change the focus of the image depending on distance. So the eyes need to focus on a fixed distance while you have them on---typically they are set up so that you focus at infinity, but it's also possible to put a diopter adjustment dial on the headset so that you can set the distance yourself, in order to use the headset without wearing glasses if you are nearsighted. (Adding a dial is one more moving part, though, so I think the big consumer brand headsets don't have it in order to save cost.)

      This is actually a problem, because in real life humans use both the binocular verge and the focus distance as cues to infer the depth of objects, and in VR you only get the verge. Different people rely more on the focus or on the verge, so for some it's not an issue, but some people feel it is uncomfortable and tiring, see The Obscure Neuroscience Problem That's Plaguing VR.

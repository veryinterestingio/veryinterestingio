      • Rubcionnnnn

        There is a lot of misinformation about how cores work in a computer. There are lots of tasks that are extremely difficult to enhance by using more cores, most notably real time calculations like gaming requires. Having more cores won't help speed up a task if that task requires a step by step calculation that can't be calculated if if previous step isn't finished.

        An example of this would be like having one person stack bricks as high as possible. You can add an extra person to add bricks while the other person is busy grabbing another, but adding 15 more people won't help because they can't put a brick on the stack while someone else is in middle of doing so.

      • askbow

        Also, it overshoots on purpose on a hysteresis loop.

        The physical values fluctuate all the time, and the sensors are unprecise. So if we target exactly 70, we would be turning the cooler/heater on-off all the time. This tends to break things (an expensive AC, from that perspective, is not so different from a light bulb).

        So a calculation of over- and under- shoot values goes into the design to prevent that, while maintaining the system somewhere around the target value.

    • thetomahawk42

      Going from Room A to Room B, where A is slightly warmer than B, will result in B feeling cold. Going from room C to that same room B, where C is slightly colder than B, will result in B feeling warm.

      So if the A/C is on to cool down the room, it will feel colder as the temperature is dropping. Also if you are in the flow of the air it will feel very cold as the incoming air will be far below the temperature of the room.

    • jarfil

      You can run some "conditionals" as math operations, like if you have a series of numbers [1,2,3,4] and you multiply it by another series of numbers [0,1,1,0] to get [0,2,3,0] so it's like "return from the first list if the second is 1, otherwise return 0".

      But more often you run your conditionals on the CPU, and once you've made the decisions about which operations to run on which data, you send the data in bulk with a list of fixed operations to run on it to the GPU.

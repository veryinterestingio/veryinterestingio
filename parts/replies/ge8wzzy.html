      It's because the unsubscribe link behaves differently in Europe.

      Before GDPR, when you clicked unsubscribe, you then were directed to a webpage where you'd make the changes.

      Now, in Europe and according to GDPR, when you click unsubscribe, in order for the website to be compliant no extra settings need to be changed, the unsubscribe action needs to be one click.

      Using a VPN might trick the unsubscribe website to apply the GDPR rules and unsubscribe without extra stuff like "only from this kind of email but not from all others, we'll still send those".

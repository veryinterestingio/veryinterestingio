      So here's a fun side fact. All computers count time using the number of seconds from midnight on January first 1970. This is called Unix Time and the second the clock struck twelve is referred to as The Epoch. So when your battery died it essentially reset to 0 which according to the computer is 1970. Another fun fact is that at the time of this posting the current Unix Time is 1622505116 which means that many seconds have passed since 1970

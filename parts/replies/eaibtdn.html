    Polish is the same (not surprisingly, I guess), and when you're a native speaker you don't really think about it, but the results can be really weird.

    For example, when used as a person's name, "Mercedes" will obviously be considered feminine, but when used to describe a car, it becomes a masculine noun.

    Edit: For extra craziness, this of course doesn't hold for all car names. A Toyota would be feminine, a BMW neuter. (on the other hand, a slangy name for a BMW, "beemka" (be-em-ka), is feminine...)

      Welp, surprisingly enough, that's also the rules in Hebrew. Which is not only not a Slavic Language, but also not an Indo-European one. Most of the time, if a word ends with an "-a" sound, it'll be feminine (and end with ה, which is basically a script-transition induced silent H). If it won't, it'll be masculine. Nothing for netuer because it's not a thing here.

      For example, "אִינסְטַגרַם" (Instagram) would be masculine, while "אַלפָּקָה" (Alpaca) would be feminine.

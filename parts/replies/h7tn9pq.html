      It's important to note here what R0 actually means. In the very simple modeling you are referring to, it means the average number of infections a person will generate in an immunologically naive community. Used this way, R0 is not a fixed number and cannot be used across diseases and across time periods to compare relative contagiousness, because the average number of infections a person will generate in an immunologically naive community depends on how that person interacts with the community. In a society where interpersonal contact is very rare, R0 will be much lower for the exact same disease than a society where interpersonal contact is frequent and extensive. And the herd immunity threshold can also, therefore, be highly variable between cultures, and vary within the same culture if and when the culture changes the way it interacts (for example, as the result of the epidemic disease).

      In other words, the herd immunity threshold is a different value, for example, in a society with universal masking indoors than it is in a society with no masking at all. It is a different value in a society with extensive hand-washing and physical distancing than a society with less hand washing that is not as concerned about physical distancing. And if the herd immunity threshold for universal masking is reached and cases begin to decline, and masking behavior then changes, cases could begin to rise again because the different behavior means a higher threshold of immunity is required to control the disease, as it is now easier for the disease to spread from person to person.

    • Jiquero

      In the ideal mathematical situation, the maximum proportion of non-immune people allowed is inversely proportional to the infectiousness.

      If R0=2, then 50% need to be immune; if R0=4, then 75%; if R0=5 then 80%, etc.

      (Of course in real life, immunity isn't a binary thing, and the infectiousness varies with circumstances so much that an exact value isn't a good model.)

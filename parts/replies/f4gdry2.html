    Also like to point out that the "Raw uncompressed master" also has all/many of the compression techniques applied to it, so no masters are truly compressed in a "every pixel of every frame has a perfect RGB value assigned" kind of way. Just that it has not been compressed as much as a Bluray/streaming file would.

    Technically, the only uncompressed master would be if it were archived on film.

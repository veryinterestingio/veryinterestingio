    The 6502 had direct memory access for most operations, so you could say

    load value1, $4002 ; memory address 4002. Address that stores y

    add value1, #5 ; actual value 5. The number to add

    store value1, $4010 ; memory address 4010. Address that stores x

    Of course that's not actual 6502 assembly, but pseudocode for the operations.

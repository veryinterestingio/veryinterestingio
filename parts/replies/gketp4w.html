      Rocket tanks usually have vents so that if overpressure is an issue, they can vent the excess pressure. Usually because a launch has been scrubbed, and liquid fuels are cryogenic, so need to be removed before they boil/expand.

      And sudden overpressures are usually a sign that something very bad has happened inside the tank (see: AMOS-6), so even if you put in a very overdesigned tank inside, the mission would need to be aborted anyway. and overdesigning a tank so that it wouldn't just burst from such a shock would make the rocket too heavy to fly.

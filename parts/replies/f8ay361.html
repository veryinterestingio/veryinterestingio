    • BuckNZahn

      It‘s usually 4K or 8K, depending on the theatre. But what makes the size difference is not the resolution, but the format and the difference in bitrate.

      For a blueray, the movie will be compressed so the data can fit on the disk. The theatre will get the uncompressed file.

      You can google video compression for examples how making the file size smaller works. Basically, you try to find patterns in the infomartion that can be described more efficiently. You also ignore small differences to make larger patterns.

      Basically, you sacrifice some details in the information, in order to get nicer patterns, which then can be saved using less space.

      Edit: See comments to this post for corrections of innacruacies.

      • kin0025

        Theatres get compressed video, just losslessly compressed. No data is lost, but if patterns can be found in the original data they will still be used. Otherwise movies would be ludicrously large - for 4K 24fps, 10 bit colour movies you're looking at nearly 6Gbps, or 800MB/s. That's 5.4 Terabytes for a 2 hour movie, which can't be played back - HDDs have the capacity but can't read data fast enough, and SSDs are far too expensive and not typically available in such high capacities.

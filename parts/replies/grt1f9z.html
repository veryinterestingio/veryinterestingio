    • Lower-Wallaby

      I have been involved in rallying from club to interantional level, so i will say that you dont need a rear wiper.

      It is a very japanese thing to have a rear wiper. But they arent actually on the homologation papers or class rules for rally cars so they get removed.

      Example is a Ford Fiesta R2 which is a hatch comes with a rear wiper standard, but the R2 rules says you can remove it. But if the photos are there and there is no overising rule or homologation update you need it.

      Funniest thing though, it was either the evo 7 or 8 that had the airconditioner still present in the homologation photos, so if you ran group N then you had to have an airconditioner. Didnt need to work, but had to be there.

      So TLDR, not a rally thing, a japanese thing

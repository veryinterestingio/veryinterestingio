    • Many-Body-Problem

      The problem is that if you would want to build a reference kilogram to use in your scales you will need to count out exactly x protons. For the second that's not so difficult because atomic clocks do essentially that, but for protons that's a lot harder, so you get a big uncertainty on your reference kilogram. The more complicated definition is just easier to implement to a high accuracy in practice.

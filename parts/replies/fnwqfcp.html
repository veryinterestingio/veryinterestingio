    • alcaizin

      As far as I know it's less about the language that it's written in and more about how the operating system functions and what interface it provides to the hardware resources. If the game happens to use features/parts of the interface that are still essentially the same, it'll still run fine. If it uses features that are outdated but compatibility has been maintained because they were commonly-used, it'll probably be fine as well. If it uses features that have been removed or re-written to have a different interface, then it won't be able to do what it needs to do and will likely crash.

    • greenlepricon

      You might get some surprising results if you open the windows and run some fans in the basement. You'll really want some airflow to move the radon along. I'm not an expert at radon in homes and usually work in natural environments, so I don't want to prescribe a solution to your home specifically, but you're essentially doing the same thing as the ventilation system. The biggest advantages to venting the slab are that you don't have to worry about outdoor/indoor temperatures, which might not be an issue depending on where you live, and it's nice to plug in and forget about. If those don't make the cost worth it to you then absolutely open the windows and circulate the air.

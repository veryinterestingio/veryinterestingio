    • Sleepinator2000

      Each studio has a different definition, but generally the idea is to have all teams be able to confidently staff up and start producing the content for the game. Each group has a slightly different focus:

      • Design - Core gameplay loops and systems demonstrable, either in prototype engine or in draft. Story outline to inform character style, writing, and environment creation.
      • Engineering - Core production tools and technologies in a usable state for the content teams (environment art, character art, animation, UI, audio, VFX, etc.). Ideally at least one sample of every asset has been run through a tools pipeline.
      • Art - Samples of environments, characters, animations, audio, VFX (ideally built using the prototype tools).
      • Production - Schedule, budget, and staffing plan for creation of the engine, assets, and testing roadmap.

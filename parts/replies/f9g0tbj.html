      It's the result of poor code quality or environmental factors that cause programs to use mass amounts of memory they aren't actually using. Many things can cause it .

      A big cause is programmers not deleting data correctly - if they for example remove the actual but not the address in memory or every other part, you are left with random bits of data. Add this up enough and it becomes a problem.

  • mutantenzyme

    It is not always possible to bring back extinct species via cloning in the absence of any genome editing. I highly recommend the 2015 book How to Clone a Mammoth by Beth Shapiro, which explains the idea of using CRISPR/Cas9 genome editing to modify the DNA of the elephant genome so that it contains selected genes that confer the key characteristics of mammoths. I have not read all of this book, but I think that it is a great resource that may address the issue of the risks of inbreeding within initially small populations of previously extinct species. Interestingly, the last living mammoths may have become extinct due to inbreeding, so it is a concern worth thinking about. Some extinct species may be better candidates than others for de-extinction; there are a number of issues (are cells preserved? can the DNA sequence be obtained even if no intact cells remain?). Smithsonian magazine mentions the idea of using genome-editing techniques to re-engineer lost genetic diversity into living populations of endangered species; this method would require DNA sequences from multiple specimens.

      What's the probability that n hays doesn't break the camel's back?

      The weight of n hays is represented by 1 point in the n-dimensional cube, randomly chosen. The sum <1 is the tetrahedron with vertices at points with a single coordinates being 1. For example, if n=3, then we have a random point inside the cube with length 1, and the tetrahedron we need is the one with vertices at (1,0,0),(0,1,0),(0,0,1).

      The volume of this tetrahedron is 1/n!.

      So the probability that n hays break the camel back is 1-1/n!.

      So the probability that EXACTLY n hays break the camel back is (1-1/n!)-(1-1/(n-1)!)=(n-1)/n!, but this formula only work for

      So the expected number of hays that break the camel back is the sum of all terms of the form n(n-1)/n!=1/(n-2)! (for n at least 2). This sum is e.

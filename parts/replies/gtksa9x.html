    The AI/ML/computational method guides our robotic experimentation. It will become more and more important in the future. For example, we don’t want to run every possible experiment using our robots.  We only want to run the valuable ones based on the output of AI/ML results, because while the robots are efficient, the experiments still take time to run.  The key is using good Design of Experiments Theory.

    You can’t minimize the unique, powerful, and beautiful resource that IS the human brain.  Our final long-term target will be an autonomous robotic experimentation process working side by side to support the researcher.

        "e," approximately 2.71828, is a number that pops up a LOT in math. In this case, OP has noticed that when you cut up any positive number into a set of (positive, to prevent cheating) numbers which add up to that number, if you multiply that set then you will get the biggest result by making the numbers you multiply as close to e as you can.

        For example, 30 (which is very close to, but slightly larger than 11e.)
        30=10         3
        10^3= 1000, but we get a much bigger number with
        3^10= 59049.
        e^11 is bigger though, about 59874.

        Once your number is smaller than e the result gets smaller too, for example 2^15 =32768.

        OP is asking for a proof to explain this.

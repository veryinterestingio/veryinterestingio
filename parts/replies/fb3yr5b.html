    A googol is so big that there aren't enough atomic particles available in the observable universe to assign to each number. The universe has around 10^(82) particles in it, and a googol is 10^(100).

    We'd need to go out into the multiverse to even consider anything near enough particles for a googolplex. Assuming, of course, that the other 10\^10\^99+ other universes that we pick are similar to ours...

    (And let's not mention Graham's Number. Or Tree(3).)

      • denny31415926

        It relates to a game you can play using n differently colored seeds. You then use the seeds to make graphs (a set of lines connecting vertices). TREE(n) is the number of graphs you can make with n differently colored seeds, such that no graph is a subset of another graph.

        This sequence grows absurdly fast. I don't remember exactly what TREE(1) and TREE(2) are, but they're less than 5. TREE(3) is a number beyond all human comprehension. There is no notation that exists that can be used to write it, even using every available atom in the universe for one symbol (eg. Googolplex is enormous but you can write it as 10^10^100 ).

    escaloni A:

    It can be more complex, but keeping it simple:

    Repeater: Takes rf signal and repeats the signal. Generally has to be same channel, and can only repeat the signal as well as it can hear it. Will add latency to connection.

    Extender: Same principal as repeater, but operates on a separate channel. Has to be in good hearing range of main rf signal or it wont work efficiently. Can have a different ssid broadcasting while still being the same network. Uplink can be wired or wireless.

    Bridge: Connects (bridges) wired infrastructure w/ a wireless (point-2-point) connection.

    PacoWaco88 A:

    https://www.faceofit.com/bridge-vs-repeater-vs-extender/amp/

    Repeaters, repeat. One listens to the core wifi network and sends the signal to device 2. Device 2 then acts like another access point you can connect to on the same network as the core network. Note, this is NOT an amplified signal or an amplifier, unless your equipment explicitly says so. That is the job of an amplifier. This signal has merely been repeated.

    An extender is a bit confusing to interpret based off of the name. It works like a repeater, however the key distinction is that device 2 creates a new network you connect to.

    A wifi bridge connects wifi-incompatible devices. Devices connect to the bridge via hard line and the bridge communicates with the wireless router. The connection to a network is via the bridge, and the bridge talks to the router on your behalf.

    thebserge A:

    To make it simple, a wireless repeater enhances the range of your wireless access point (AP) by retransmitting all of its radio signals.

    A wireless extender is the same thing with a different name, although it usually implies that it retransmits the original WiFi network name (SSID), too. A wifi repeater can also be set up like that, or you can create a new SSID.

    Wifi bridge is slightly different. It is meant to connect two or more wired networks via WiFi. Like building a bridge between two islands heh. But in practice, it is trivial to create a Wifi Bridge that also works as a wifi repeater.

    ELI5: Why are the letters x, y and z used for representing points in a coordinate system?

    grekhaus A:

    Rene Descartes started using those letters to represent unknown coordinates in his maths book 'La Geometrie' and everybody else followed the convention from there. Same reason as that the ratio between circumference and diameter uses the Ï€ symbol - Euler started doing it and it got popular from there.

    member_of_the_order A:

    X was chosen long ago as the standard variable (there's an interesting TED talk explaining why if you want me to go find it). When we need more variables for plotting on a 2-D chart or in a 3-D space, logically we just choose the next letters in the alphabet.

    Obviously we run out, so we can't quite continue the pattern with 4 spatial dimensions ("spatial" is important because if it's time we just use t), so we go backwards from x and use w.
